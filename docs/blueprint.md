# **App Name**: Lineage Kernelcrafter

## Core Features:

- Project Setup: Select target device (SM8550) and LineageOS version (23.2), and manage kernel source paths.
- Kernel Patching: Automated application of 'susfs4ksu' and 'KernelSU' patches to the kernel source.
- Kernel Compilation: Initiate the kernel build process with user-selected configurations and flags.
- Flashable Package Creator: Generate a flashable ZIP or image package suitable for device flashing.
- Build Log Viewer: Display real-time compilation output and error messages during the build process.
- AI Build Troubleshooting Assistant: A generative AI tool that analyzes build logs to identify common errors and suggests potential solutions or next steps.

## Style Guidelines:

- Primary interactive color (buttons, selected states): Neon Green (#00FF55), symbolizing 'active' and 'go'.
- Background color: Deep forest green with low saturation (#161D18), providing a dark, focused development environment.
- Accent color (highlights, warnings, special elements): Vibrant Purple (#AA00FF), for contrast and to draw attention.
- Headlines and short text: 'Space Grotesk' (sans-serif) for a modern, tech-oriented feel.
- Body text: 'Inter' (sans-serif) for readability in longer descriptions and configuration details.
- Code and log output: 'Source Code Pro' (monospace) for clear display of technical information.
- Geometric and angular icons with a metallic or digital sheen, aligning with a tech-savvy and industrial aesthetic.
- Clean, structured layouts with clearly defined panels for input, output, and progress indicators, prioritizing functional efficiency.
- Subtle progress animations and state changes, like pulsating indicators or smooth panel transitions, to provide user feedback without distraction.
